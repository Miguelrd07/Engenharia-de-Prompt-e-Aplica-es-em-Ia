“Faça um código em Python que calcule a média de notas.”

🔹 Problemas desse prompt
Não define quem está pedindo (persona)
Não define nível do código
Não define quantas notas
Não define formato da resposta
Não define se deve explicar

👉 Resultado: resposta genérica, pouco útil para iniciante.
